

# AVXelerate

AVXelerate是一个高性能的数学库，专门利用AVX2指令集优化了exp, log, tanh, 和sigmoid等数学函数。本项目旨在为需要高速数学运算的应用提供有效的加速方案，充分发挥AVX2的高级计算能力。
